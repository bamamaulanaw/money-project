# Money Project

Parent repository for the **KU-Money** application.

This repository contains the **frontend** and **backend** source code in a single project structure.

---

## Project Structure

    money-project/
    ├── ku-money-app/ # Frontend
    ├── ku-money/ # Backend
    ├── LICENSE
    └── README.md

---

## Description

- **ku-money-app**  
  Frontend application of KU-Money.

- **ku-money**  
  Backend application of KU-Money.

Each directory is a standalone application with its own configuration and documentation.
